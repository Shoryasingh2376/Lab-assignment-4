# JavaScript Quiz Game

## Overview
This is a simple quiz game created using JavaScript for *Lab Assignment 4*.  
It runs entirely in the browser console and demonstrates the use of:
- Arrays
- Loops
- Conditionals
- Functions
- Basic Input/Output (prompt and alert)

## Features
- 5 predefined questions
- Input handled using toLowerCase() and trim()
- Immediate feedback after each question
- Final score displayed at the end

## How to Run
1. Open the browser console (F12 → Console tab)
2. Copy and paste the contents of quiz.js
3. Press Enter to start the quiz
